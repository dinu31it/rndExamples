# my-domo-app

> Made with create-react-library

## Install

```bash
npm install --save my-domo-app
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'my-domo-app'
import 'my-domo-app/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
